Getting Started

Welcome! This guide will help you add the widget to your website in just a few minutes

Step 1: Sign Up

Create your MyNaturevista account to get started.

1

Go to the signup page

Visit our signup page and create your account.

2

Enter your email and password

Use a strong password to keep your account secure.

3

Check your subscription email

You have received an email with the details of your subscription. Please check your inbox.

Step 2: Choose Your Plan

Select a subscription plan that fits your needs. You can upgrade or downgrade anytime.

Go to Upgrade Plan in the sidebar to choose and pay for your subscription.

Step 3: Create Your First API Key

An API key connects the widget to your account and controls which websites can use it.

1

Go to "API Access" in sidebar

Find it under the "Admin Widget" section.

2

Click "Generate Api Key"

You'll see a form to set up your key.

3

Give it a name

Use something descriptive like "My Website" or "Main Site".

4

Click Generate

Your API key will be generated and displayed. Copy it for the next step.

Step 4: Copy Your Widget Code

Now you'll get the code to add to your website.

1

Go to "Integration Guide" in sidebar

You'll find it under "Admin Widget" section.

2

Copy the script code

You'll see a code snippet with your API key already included. Click the copy button.

3

Paste it before the closing </body> tag

Add this code at the bottom of your website's HTML, just before </body>.

Step 5: Add a Button

Add a button to your website so visitors can open the widget.

1

Copy button code from Integration Guide

You'll find example button code on the Integration Guide page.

2

Paste it where you want the button

Add the button code to any page where you want visitors to access the widget.

3

Customize button text and style

You can change the button text to match your brand. See Edit Buttons for styling options.

Pre-made Buttons Available

Check out our Buttons gallery for ready-to-use button designs you can copy.

Step 6: Test It!

Make sure everything is working correctly.

1

Visit your website

Open your website in a web browser.

2

Click the button

Try clicking the button you added.

3

The widget should open

You should see the widget appear with a list of destinations.

Troubleshooting

Widget not showing?
  • Check that your API key is correct
  • Make sure your domain is added to the allowed list in API Access
  • Contact support if you need help
Button not working?
  • Check that the widget script is loaded on the page
  • Make sure the button calls openNatureWidget() function
