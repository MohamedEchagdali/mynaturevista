Widget Types Overview

MyNaturevista offers three different widgets to choose from based on your needs.

Main Widget

Shows all countries and destinations. Best for homepage or general pages.

Country Widget

Shows destinations from one specific country. Perfect for country-specific pages.

Place Widget

Shows one specific destination. Ideal for detailed destination pages.

Main Widget (All Destinations)

The main widget displays all available destinations with full browsing capabilities.

Basic Code

HTML 
<script src="https://api.mynaturevista.com/widget.html?apikey=YOUR_API_KEY" async></script>
Important

Replace YOUR_API_KEY with your actual API key from API Access page.

Country Widget (Specific Country)

Show destinations from one specific country.

Basic Code

HTML 
<script src="https://api.mynaturevista.com/widget-country.html?apikey=YOUR_API_KEY&country=Italy" async></script>

Replace YOUR_API_KEY with your key and Italy with any country name from the available countries list.

Tip

Country names are case-sensitive. Use exact names from the list below.

Single Place Widget

Display one specific destination with its details.

Basic Code

HTML 
<script src="https://api.mynaturevista.com/widget-eachPlace.html?apikey=YOUR_API_KEY&place=FranceCalanquesOfCassis" async></script>

Replace YOUR_API_KEY with your key and FranceCalanquesOfCassis with any place name from the available places list.

Custom Locations

You can also use your custom locations created in the "Custom Locations" section.

Customization Options

You can customize the widget's appearance by adding URL parameters.

Available Parameters

Parameter Description Example
name Custom button text &name=Explore
primaryColor Main color (hex format with %23 instead of #) &primaryColor=%23ff5733
secondaryColor Secondary color (hex format with %23 instead of #) &secondaryColor=%2333c1ff
fontFamily Custom font (use + for spaces) &fontFamily=Open+Sans

Full Customization Example

HTML 
<script src="https://api.mynaturevista.com/widget-country.html?apikey=YOUR_API_KEY&country=France&name=Discover%20France&primaryColor=%23ff5733&secondaryColor=%2333c1ff&fontFamily=Montserrat" async></script>
Important: Color Format

Colors must use hex format with %23 instead of #. Example: #ff5733 becomes %23ff5733

URL Encoding Quick Reference

Character URL Encoded
Space %20
# %23
& %26

Domain Setup

For security, you must register which domains are allowed to use your API key.

Adding Your Domain

1

Go to API Access

Click "API Access" in the sidebar

2

Create or Edit Key

Create a new key or edit an existing one

3

Add Domain

Enter your domain (e.g., https://mywebsite.com)

Always Use HTTPS

Always use https:// (not http://)

How Openings Are Counted

What Counts as an Opening

Action Counted?
Widget loads on page No
Visitor clicks to open widget Yes - 1 opening
Browsing inside widget No

Troubleshooting

Common Issues

Widget not showing
  • Check that you replaced YOUR_API_KEY
  • Verify domain is added to API key settings
  • Wait 2-3 minutes and refresh
"Domain Not Allowed" error

Add your domain in API Access settings
